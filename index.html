Site criado para o Projeto Final de Linguagens de Script

Esse site foi criado para demonstrar o conhecimento adquirido ao longo da disciplina de Linguagens de Script.

Funcionalidades

Através do uso de JavaScript fizemos uma maneira de inserir músicas em uma lista. Com isso é possível filtrá-las na página "Músicas" a qualquer momento, além de descobrir novas músicas que talvez você não conheça.
